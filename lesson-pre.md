# 课堂预演页面描述：

一、整体设计风格
页面背景：浅白色基底 + 淡粉色柔和渐变，全局采用玻璃态 / 毛玻璃 UI，大圆角设计，简约科技感，适配教育类 AI 产品
布局：垂直流式布局，所有核心内容水平居中，响应式适配不同屏幕尺寸
视觉规范：现代无衬线字体（如 Inter），层级清晰；主文字深灰 / 黑色，辅助文字浅灰，品牌强调色为粉色
二、页面模块结构（从上到下）

1. 品牌 Logo 区（页面顶部居中）
核心元素：课堂预演 品牌标识
左侧：几何立方体风格的双色图标（粉色+Anthropic官网同款黄色）
右侧：文字「Lesson Rehearsal」，其中「Lesson 」为深灰色 / 黑色，「Rehearsal」为粉色，加粗展示

2. 核心交互卡片（主输入区，居中，大尺寸圆角容器）
容器样式：大圆角（≥16px）、浅白色毛玻璃背景、轻微阴影，还原玻璃态视觉效果
布局：左右分栏功能区
左侧（用户输入区）
顶部：默认圆形用户头像 + 文字「嗨，同学」 + 下拉箭头
主体：大输入框区域，占位提示为灰色示例文字：
plaintext
输入你想学的任何内容，例如：
「中国历史文化发展课程预演」
「数据结构红黑树课程预演」
底部左侧：水平排列 2个功能项：附件图标、语音输入麦克风语音输入图标

底部右侧：粉色圆角按钮「确认」（白色文字）

4. 课程内容区（横幅下方，居中）
标签栏：仅保留「我的课程」标签（当前选中状态，下方有下划线高亮），完全删除「精选课程」标签
课程卡片：水平排列的课程卡片（圆角矩形、白色背景、轻微阴影）
单卡片结构：
顶部：课程标题（深色字体）
主体：左侧为课程要点 / 目录（灰色小字列表），右侧为课程封面图（主题相关视觉图 / 公式图）
底部：左侧为课程信息（页数、日期，浅粉色字体），右侧为课程名称（深色字体）
三、开发额外要求
视觉还原：精准还原圆角、渐变、阴影、玻璃态效果，色彩匹配原图（主色：粉色 + 深灰，辅助色：浅粉 / A nthropic官网同款黄色 ）
基础交互：按钮、输入框、下拉箭头等元素保留基础 hover 态效果
